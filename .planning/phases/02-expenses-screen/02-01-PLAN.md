<objective>
Deliver the first working vertical slice of the Expenses screen: the app launches directly into a Jetpack Compose screen that fetches the current month's expenses from the backend over the existing HMAC-signed connection and renders them newest-first with a total-spend summary card. (EXP-01)

Date range selection (EXP-02), category filter chips (EXP-03), and loading/error/empty states are Plan 02.

Output: A runnable APK whose launch screen shows live expense data for the current month.
</objective>
